# Comunidad Azul

Sitio web del Barrio Divino Niño 1 (Colombia): información sobre agua potable, puntos de venta, reportes y contacto.

**Sitio en vivo:** https://comunidad-azul.netlify.app  
**Repositorio:** https://github.com/SantiagoMontenegroC/comunidad-azul

## Desarrollo local

Abre `index.html` en el navegador o sirve la carpeta con cualquier servidor estático.

Configura Supabase en `js/supabase.js` y ejecuta `supabase/schema.sql` en tu proyecto.

## Despliegue

Push a la rama `main` → Netlify publica automáticamente.
